Instructions for starting a session by reviewing the non-linear Multiset in idris2-LMset and running its tests.

At the start of every session involving the Multiset logic, follow these steps:

1.  **Review the Implementation**: Check the non-linear Multiset implementation in `/var/home/justin/Projects/idris2-LMset/src/LMset.idr`.
2.  **Run Tests**: Execute the associated test suite using the following command in the `/var/home/justin/Projects/idris2-LMset` directory:
