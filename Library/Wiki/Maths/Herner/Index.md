# Eric Hehner: Unified Algebra & Probability

This folder contains summaries and notes on the work of **Eric C. R. Hehner** (University of Toronto), focusing on his innovative approaches to unifying logic, algebra, and probability.

## Documents

- **[From Boolean Algebra to Unified Algebra (BAUA)](./BAUA.md)**
  An argument for simplifying boolean algebra using lattice-based symbols ($\top, \perp, \land, \lor, \leq$) and unifying it with number algebra. It proposes that "proof" and "solution" are both forms of algebraic calculation.

- **[A Probability Perspective (ProPer)](./ProPer.md)**
  An approach that equates probability with information on a different scale ($I = -\log p$). It uses a programming formalism (assignments, weighted if-then-else) to solve classic probability paradoxes like Monty Hall and the Two Envelopes.

## Key Themes
- **Formalism**: Moving from informal argument to formal calculation.
- **Unification**: Applying the same algebraic laws across logic, numbers, and probability.
- **Scale**: Information, probability, and state space are different ways to measure the same underlying quantity.
