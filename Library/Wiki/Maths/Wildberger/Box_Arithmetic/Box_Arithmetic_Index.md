# Box Arithmetic: Comprehensive Index

This index provides a structured overview of Norman Wildberger's "Box Arithmetic" series, which redefines standard arithmetic and algebra using multisets and hierarchical structures. This framework is essential for the verification of Rational Chromogeometric Information Theory (RCIT).

## Sections

- **[01: Multiset Foundations](./01_Multiset_Foundations/Index.md)**
  Introduction to the core vision, tree-based arithmetic, and the construction of negative numbers through anti-multisets (MF 226-232).

- **[02: Polynumbers and Algebra](./02_Polynumbers_and_Algebra/Index.md)**
  Algebraic theory of integral, symmetric, and alternating polynumbers, with applications to group characters (MF 233-236).

- **[03: Box Arithmetic Applications](./03_Box_Arithmetic_Applications/Index.md)**
  Practical application of the box framework to Zeta functions, Euler products, Q-Series, and multiplicative functions (MF 237-243).

- **[04: Research and Reference](./04_Research_and_Reference/Index.md)**
  Formal research papers, combinatorics notes (Maxels), and raw transcripts.
