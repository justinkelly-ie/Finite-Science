# Project Index: Rational Physics 

This project applies rational mathematics to Stochastic Quantum Theory. The mathematics uses Norman Wildberger's formulations. The information theory is based on Eric Hehner's Scale of Information. The physics is based on Jacob Barandes' Indivisible Stochastic Processes.

This document contains references to the methodology and links to high level resources, foundational notes and findings for the project.

## Code Organisation
The RCIT ecosystem is modularized into several specialized Idris 2 projects.
- **[Project Registry](./Projects.md)**: Master list of repositories, chapters, and development status.

We are migrating to a modular design (away from [idris2-rcit](../idris2-rcit)) focused on:
- Modularity, Composability, and Formal Testability.


## Library Organisation

To maintain clarity across the project, information should be categorised according to the following directory structure:

- **[Knowledge Base](../Knowledge_Base/)**: Storage for new knowledge, refined project notes, and stabilised research findings.
- **[Unified Theory and Codebase Source](./Projects_Wiki_Source/)**: The primary unified directory containing `Theory_and_Architecture.md` (physics axioms) and `full.md` (complete source codebase dump). All AI agents MUST refer to this directory for the current active architecture.
- **[Wiki](../Wiki/)**: Research materials derived from primary sources (e.g., lecture transcripts, mathematical foundations).
- **[Source (Src)](../Src/)**: Code examples, reference implementations, and patterns of high-quality Idris 2 programming style.
- **[Raw](../Raw/)**: Unprocessed content, raw data, or documentation downloaded directly from the internet.
