::: {.callout-note title="Chapter Summary"}
**Chapter Intent:** This chapter describes Singletons  (`Sing`), Pixels (`Pix`) and Maxels (`Maxel`)- foundational types in Wildbergers Box Aritmetic, these are defined as finite collections of Multisets (`MSet1`). 

The primary focus of this library is verifying structural mathematical properties (e.g., reflexivity, transitivity, symmetry) of these objects using the dependently typed programming language Idris 2 [@brady2021idris].
:::

Welcome to the **Base module**! This chapter introduces the foundational data structures for the RCIT framework. 

As discussed by Wildberger [@wildberger2020box], we build upon the `MSet1` structures to define our metric spaces. Let us look at the core imports and our first type, `Sing`.
