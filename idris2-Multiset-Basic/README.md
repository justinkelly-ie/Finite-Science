**Chapter Intent:** This chapter implements the `MSet1` (Multiset 1) data structure, the core building block of Norman Wildberger's Box Arithmetic.

We define the algebraic operations (addition, multiplication, caret) and verify their properties using Hedgehog and Idris 2 [@brady2021idris].

Welcome to the **MSet1 module**! This chapter defines the recursive multiset structures that represent natural numbers and polynomials in Box Arithmetic.
